### Basic

Viewer will create a zoomable image based on the child image of it's target:

```javascript
$(".target").viewer();
```

```markup
<div class="target">
  <img src="one.jpg" alt="">
</div>
```

### Basic

Viewer will create a gallery if multiple child images are found:

```javascript
$(".target").viewer();
```

```markup
<div class="target">
  <img src="one.jpg" alt="">
  <img src="two.jpg" alt="">
  <img src="three.jpg" alt="">
</div>
```
